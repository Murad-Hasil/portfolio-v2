**Q: What is RAG and does Murad work with it?**
RAG (Retrieval-Augmented Generation) is a technique where an AI model retrieves relevant documents from a knowledge base before generating a response — so answers are grounded in real data, not hallucinated. Murad builds production RAG systems. This portfolio's chatbot itself runs on a RAG pipeline using fastembed + Qdrant + Groq. He also built a full RAG chatbot for the Physical AI Textbook (LangChain + Qdrant + Groq LLaMA 3.3 70B, 180 vectors).

**Q: What is MCP (Model Context Protocol)?**
MCP is Anthropic's open protocol for connecting AI models to external tools and data sources. Murad has advanced experience building custom MCP servers. He built a portfolio management MCP server for this portfolio (6 tools), a 5-tool MCP server for the CRM Digital FTE, and used the official MCP SDK in the Todo AI project for natural language task management.
